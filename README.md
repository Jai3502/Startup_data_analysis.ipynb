# Startup Dataset Analysis 

##  Project Overview
This project performs exploratory data analysis (EDA) on a startup dataset using Python.  
The goal is to understand key patterns, trends, and insights from startup-related data through statistical analysis and visualizations.

This project is suitable for:
- Python learning students
- Data analysis practice
- Academic assignments
- Beginner data science projects

---

##  Dataset
The dataset contains information related to startups and their attributes.  
It is used to analyze patterns such as growth trends, funding insights, and other relevant factors.

---

##  Technologies Used
- Python 
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

##  Features
- Data loading and inspection
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using graphs and charts
- Insight generation from data
